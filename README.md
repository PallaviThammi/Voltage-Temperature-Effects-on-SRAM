# Voltage-Temperature-Effects-on-SRAM
This project analyzes the behavior of a 6T SRAM cell using LTspice.
## Objective
To study the effect of supply voltage on SRAM operation using voltage sweep analysis.
## Features
- SRAM voltage analysis
- VDD parameter sweep
- Waveform observation using LTspice
## Software Used
- LTspice
## Files Included
- SRAM voltage analysis.asc
- SRAM_voltage_analysis.png
